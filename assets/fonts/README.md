# Fonts Directory

## Neue Haas Grotesk Display Pro

This is a commercial font. You need to:

1. **Purchase/Download** the font from:
   - https://www.linotype.com/fonts/neue-haas-grotesk-family
   - Or use Adobe Fonts if you have access

2. **Convert to web formats** (if needed):
   - Use a service like https://transfonter.org/ to convert to WOFF2/WOFF

3. **Place the following files here**:
   - `NeueHaasDisplayPro-Regular.woff2`
   - `NeueHaasDisplayPro-Regular.woff`
   - `NeueHaasDisplayPro-Medium.woff2`
   - `NeueHaasDisplayPro-Medium.woff`
   - `NeueHaasDisplayPro-Bold.woff2`
   - `NeueHaasDisplayPro-Bold.woff`
